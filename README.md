# Student Attendance Management System

## Description

This project is a Student Attendance Management System developed to manage and track student attendance efficiently. It helps in recording daily attendance, storing data, and viewing attendance details easily.

## Features

* Add and manage student details
* Mark daily attendance
* View attendance records
* Update and delete student data
* Simple and user-friendly interface

## Technologies Used

* Java
* (Spring Boot / Core Java / Thymeleaf / Maven Repository)
* REST API
* MySQL
* HTML, CSS,JS

## How to Run

1. Download or clone the repository
2. Open the project in your IDE (Eclipse / IntelliJ)
3. Configure database (if applicable)
4. Run the application

## Purpose

This project is created for learning purpose to understand real-time application development and improve programming skills.
